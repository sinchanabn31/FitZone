# FitZone - Animated Fitness Website

FitZone is a modern, responsive, and animated fitness website built using **HTML, CSS, and JavaScript**. The website is designed to provide an attractive user experience with smooth animations, interactive sections, and a professional gym-themed interface.

## 🌐 Live Demo

👉 https://sinchanabn31.github.io/FitZone/


## ✨ Features

- 🏠 Responsive Navigation Bar
- 🎬 Animated Hero Section
- 💪 About FitZone Section
- 🏋️ Fitness Programs
- 👨‍🏫 Professional Trainers Section
- 💰 Membership Pricing Plans
- ⭐ Customer Testimonials
- 📞 Contact Section
- ⬆️ Smooth Scrolling
- 🎨 Hover Animations
- 📱 Mobile Responsive Design
- 🚀 Animated UI Components

1. Clone the repository

```bash
git clone https://github.com/sinchanabn31/FitZone.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.
